README

I) Как запустить?

	1) Скачать все файлы проекта и разместить их в одной папке
	2) С помощью менеджера пакетов pip выполнить команду pip install -r requirements
	3) Запустить файл "Show_neural_network.ipynb" программой, способоной открывать файлы формата .ipynb, например
	jupyter notebook

Эта программа создана для ознакомления с тем, как происходит процесс обучения нейронной сети.

II) После запуска:

	Инструкция по использованию:
	1) В поле "Architecture" нужно ввести n чисел через пробел, где каждое число означает количество нейронов в текущем слое,
	а номер этого число отображает номер слоя. ПЕРВОЕ ЧИСЛО ДОЛЖНО БЫТЬ 5, а ПОСЛЕДНЕЕ 1, т.к. в данной версии нейронная сеть
	обучается на конкретном наборе данных.

	2) После того, как вы ввели  корректное значение в поле "Architecture", нажмите "submit". Вы увидите нейросеть заданной
	вами архитектуры.

	Что вы видите перед собой?
	Вертикальная последовательность кружочков- слой нейронной сети
	Кружочек- один нейрон
	Линии, соединяющие нейроны- веса нейронной сети.
	Цвета линий означают: черный- отрицательный вес, белый-положительный вес.

	3) Далее выберите один из режимов:
		"make_one_iteration"-в этом режиме после каждого нажатия кнопки "make_one_iteration" происходит одна эпоха обучения нейросети.
		"train"-в этом режиме после нажатия кнопки "train" нейросеть последовательно обучается, пока не будет нажата
		кнопка "Stop"
			Кнопка "Stop" позволяет остановить обучение нейросети.
		И в том и в другом режимах справа будет отображаться номер итерации и значение функции ошибки для данной итерации.

	4) После начала обучения в любом из режимов через некоторое время вы увидите, что толщина линий изменилась. Изменение толщины
	линий отображает изменение весов нейронной сети.
 
Все обучение производится на датасете "titanic", ознакомиться с которым можно по ссылке https://www.kaggle.com/competitions/titanic
